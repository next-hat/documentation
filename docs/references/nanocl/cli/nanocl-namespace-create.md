# Namespace create

# NAME

create - Create new namespace

## SYNOPSIS

**create** \[**-h**\|**--help**\] \<*NAME*\>

## DESCRIPTION

Create new namespace

## OPTIONS

**-h**, **--help**  
Print help

\<*NAME*\>  
name of the namespace to create
