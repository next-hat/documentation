# Vm remove

# NAME

remove - Remove vms

## SYNOPSIS

**remove** \[**-y**\|**--skip-confirm**\] \[**-h**\|**--help**\]
\[*KEYS*\]

## DESCRIPTION

Remove vms

## OPTIONS

**-y**, **--skip-confirm**  

**-h**, **--help**  
Print help

\[*KEYS*\]  
The keys or names of the objects to remove
