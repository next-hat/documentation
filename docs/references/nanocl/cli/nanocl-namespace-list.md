# Namespace list

## SYNOPSIS

**list** \[**-q**\|**--quiet**\] \[**-l**\|**--limit**\]
\[**-o**\|**--offset**\] \[**--filters**\] \[**-h**\|**--help**\]

## DESCRIPTION

List existing namespaces

## OPTIONS

**-q**, **--quiet**  
Only show keys

**-l**, **--limit**=*LIMIT*  
Limit the number of results default to 100

**-o**, **--offset**=*OFFSET*  
Offset the results to navigate through the results

**--filters**=*FILTERS*  
Filters

**-h**, **--help**  
Print help
