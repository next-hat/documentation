# Secret patch env

# NAME

env - Update env secret values

## SYNOPSIS

**env** \[**-h**\|**--help**\] \<*VALUES*\>

## DESCRIPTION

Update env secret values

## OPTIONS

**-h**, **--help**  
Print help

\<*VALUES*\>  
List of values in the form of \`key=value\`
