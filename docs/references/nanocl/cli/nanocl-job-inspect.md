# Job inspect

# NAME

inspect - Inspect a job by its name

## SYNOPSIS

**inspect** \[**--display**\] \[**-h**\|**--help**\] \<*KEY*\>

## DESCRIPTION

Inspect a job by its name

## OPTIONS

**--display** *\<DISPLAY\>*  
Display format  

  
\[*possible values: *yaml, toml, json\]

**-h**, **--help**  
Print help

\<*KEY*\>  
Key or Name of the object to inspect
