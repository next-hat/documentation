# Resource history

# NAME

history - Browse history of a resource

## SYNOPSIS

**history** \[**-h**\|**--help**\] \<*NAME*\>

## DESCRIPTION

Browse history of a resource

## OPTIONS

**-h**, **--help**  
Print help

\<*NAME*\>  
The name of the resource to browse history
