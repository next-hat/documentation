# Event

## SYNOPSIS

**event** \[**-h**\|**--help**\] \<*subcommands*\>

## DESCRIPTION

Show or watch events

## OPTIONS

**-h**, **--help**  
Print help

# SUBCOMMANDS

event-list(1)  
List existing events

event-watch(1)  
Watch for new events in real time

event-inspect(1)  
Inspect a specific event

event-help(1)  
Print this message or the help of the given subcommand(s)
