# Namespace inspect

## SYNOPSIS

**inspect** \[**--display**\] \[**-h**\|**--help**\] \<*KEY*\>

## DESCRIPTION

Inspect a namespace

## OPTIONS

**--display**=*DISPLAY*  
Display format  

  
\[*possible values: *yaml, toml, json\]

**-h**, **--help**  
Print help

\<*KEY*\>  
Key or Name of the object to inspect
