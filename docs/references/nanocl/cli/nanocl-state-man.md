# State man

# NAME

man - Display documentation for a statefile

## SYNOPSIS

**man** \<**-s**\|**--source**\> \[**-h**\|**--help**\]

## DESCRIPTION

Display documentation for a statefile

## OPTIONS

**-s**, **--source** *\<SOURCE\>*  
Path or URL to the statefile

**-h**, **--help**  
Print help
