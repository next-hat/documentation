# Vm image clone

## SYNOPSIS

**clone** \[**-h**\|**--help**\] \<*NAME*\> \<*CLONE_NAME*\>

## DESCRIPTION

Clone a VM image

## OPTIONS

**-h**, **--help**  
Print help

\<*NAME*\>  
Name of the VM image

\<*CLONE_NAME*\>  
Name of the cloned VM image
