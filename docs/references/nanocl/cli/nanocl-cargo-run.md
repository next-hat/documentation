# Cargo run

## SYNOPSIS

**run** \[**-v**\|**--volume**\] \[**-e**\|**--env**\] \[**--rm**\]
\[**-h**\|**--help**\] \<*NAME*\> \<*IMAGE*\> \[*COMMAND*\]

## DESCRIPTION

Run a cargo

## OPTIONS

**-v**, **--volume**=*VOLUMES*  
Volumes of the cargo

**-e**, **--env**=*ENV*  
Environment variables of the cargo

**--rm**  

**-h**, **--help**  
Print help

\<*NAME*\>  
Name of the cargo

\<*IMAGE*\>  
Image of the cargo

\[*COMMAND*\]  
Command to execute
