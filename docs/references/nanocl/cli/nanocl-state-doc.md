# State doc

# NAME

doc - Display documentation for a statefile

## SYNOPSIS

**doc** \<**-s**\|**--source**\> \[**-h**\|**--help**\]

## DESCRIPTION

Display documentation for a statefile

## OPTIONS

**-s**, **--source** *\<SOURCE\>*  
Path or URL to the statefile

**-h**, **--help**  
Print help
