# Secret

## SYNOPSIS

**secret** \[**-h**\|**--help**\] \<*subcommands*\>

## DESCRIPTION

Manage secrets

## OPTIONS

**-h**, **--help**  
Print help

# SUBCOMMANDS

secret-remove(1)  
Remove existing secret

secret-list(1)  
List existing secret

secret-inspect(1)  
Inspect a secret

secret-create(1)  
Create a new secret

secret-help(1)  
Print this message or the help of the given subcommand(s)
