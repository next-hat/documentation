# Metric

## SYNOPSIS

**metric** \[**-h**\|**--help**\] \<*subcommands*\>

## DESCRIPTION

Manage metrics

## OPTIONS

**-h**, **--help**  
Print help

# SUBCOMMANDS

metric-list(1)  
List existing metrics

metric-inspect(1)  
Inspect a metric

metric-help(1)  
Print this message or the help of the given subcommand(s)
