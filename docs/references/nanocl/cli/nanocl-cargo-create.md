# Cargo create

# NAME

create - Create a new cargo

## SYNOPSIS

**create** \[**-v**\|**--volume**\] \[**-e**\|**--env**\]
\[**-h**\|**--help**\] \<*NAME*\> \<*IMAGE*\>

## DESCRIPTION

Create a new cargo

## OPTIONS

**-v**, **--volume** *\<VOLUMES\>*  
Volumes of the cargo

**-e**, **--env** *\<ENV\>*  
Environment variables of the cargo

**-h**, **--help**  
Print help

\<*NAME*\>  
Name of the cargo

\<*IMAGE*\>  
Image of the cargo
