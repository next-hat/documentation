# Cargo patch

## SYNOPSIS

**patch** \[**-n**\|**--name**\] \[**-i**\|**--image**\]
\[**-e**\|**--env**\] \[**-v**\|**--volume**\] \[**-h**\|**--help**\]
\<*NAME*\>

## DESCRIPTION

Update a cargo by its name

## OPTIONS

**-n**, **--name**=*NEW_NAME*  
New name of cargo

**-i**, **--image**=*IMAGE*  
New image of cargo

**-e**, **--env**=*ENV*  
New environment variables of cargo

**-v**, **--volume**=*VOLUMES*  
New volumes of cargo

**-h**, **--help**  
Print help

\<*NAME*\>  
Name of cargo to update
