You can grab the nanocl tarball directly from the <a href={vars.nanoclReleaseTagUrl}>GitHub releases page</a>.<br/>
You can also use the following commands to download and extract the tarball:

<Terminal language="sh" filename="sh">
{`curl -LO ${props.tarballUrl}
tar -xzf ${props.tarballName}
`}
</Terminal>

<p>Extracted directory layout:</p>
<DirTree variant="unix" />

We now need to move the `nanocl` binary and it's man pages to appropriate locations.

<Terminal language="sh" filename="sh">
{`sudo mv ./bin/nanocl /usr/local/bin
sudo mv ./share/man/man1/* /usr/local/share/man/man1`}
</Terminal>
