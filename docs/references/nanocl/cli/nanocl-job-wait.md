# Job wait

# NAME

wait - Wait for a job to finish

## SYNOPSIS

**wait** \[**-c **\] \[**-h**\|**--help**\] \<*NAME*\>

## DESCRIPTION

Wait for a job to finish

## OPTIONS

**-c** *\<CONDITION\>*  
State to wait

**-h**, **--help**  
Print help

\<*NAME*\>  
Name of job to wait
