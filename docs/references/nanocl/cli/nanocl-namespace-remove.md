# Namespace remove

## SYNOPSIS

**remove** \[**-y**\|**--skip-confirm**\] \[**-h**\|**--help**\]
\[*KEYS*\]

## DESCRIPTION

Remove a namespace

## OPTIONS

**-y**, **--skip-confirm**  

**-h**, **--help**  
Print help

\[*KEYS*\]  
The keys or names of the objects to remove
